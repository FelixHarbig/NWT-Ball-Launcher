# Raspberry Pi Tests

Test scripts for Raspberry Pi (tested on Pi 5, 4GB).

These tests demonstrate basic functionality used in the main project.

## Tests

- [`camera-stepper/`](camera-stepper/) - Person tracking with camera and stepper motor control
- [`person-tracking/`](person-tracking/) - Basic person detection from camera feed
- [`servo/`](servo/) - Servo motor control test
- [`stepper-motor/`](stepper-motor/) - Stepper motor control test
